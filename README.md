## Social Music

Uma rede social para quem gosta de música, ainda em desenvolvimento.


## Tecnologias usadas

>  ReactJs

>  NextJs

>  HTML

>  CSS

>  JavaScript

## Clone ou dê um Fork


<code> git clone https://github.com/LuciLua/socialMusic.git</code>

<code> cd ./socialMusic</code>

<code> yarn </code>

<code> yarn dev </code>

## Build

<code> yarn build </code>

<code> yarn start </code>
